# Manual de Usuario - Nementium.ai

## Introducción

Nementium.ai es una aplicación de gestión fiscal y contable para autónomos y pequeñas empresas. Permite gestionar facturas, ventas, y preparar los modelos tributarios de manera automática.

## Secciones de la Aplicación

### Dashboard

El Dashboard es la pantalla principal que muestra un resumen de tu situación financiera:

- **Resumen de los últimos 6 meses**: Gráfica comparativa de ingresos vs gastos
- **Total de ventas**: Importe total de ventas en el período
- **Total de gastos**: Importe total de facturas de gastos
- **Resultado neto**: Diferencia entre ingresos y gastos

El Dashboard se actualiza automáticamente al cargar la página. Puedes hacer clic en "Dashboard" en el menú para refrescar los datos.

### Subir Factura/Ventas

Esta sección permite cargar documentos de facturas o archivos de ventas:

**Para facturas:**
1. Arrastra un archivo PDF o imagen de la factura al área de carga
2. O haz clic para seleccionar el archivo
3. La IA analizará automáticamente la factura y extraerá los datos
4. Revisa los datos extraídos y confirma o corrige
5. Guarda la factura en el sistema

**Para ventas (Amazon, eBay, etc.):**
1. Descarga el informe de ventas de tu plataforma (formato CSV)
2. Sube el archivo en la sección de ventas
3. El sistema procesará automáticamente las transacciones

Formatos soportados: PDF, JPG, PNG, CSV

### Estado Trimestral

Muestra el estado de tus obligaciones fiscales por trimestre:

- **Modelo 303 (IVA)**: Declaración trimestral del IVA
- **Modelo 130 (IRPF)**: Pago fraccionado del IRPF para autónomos
- **Estado de cada modelo**: Pendiente, En preparación, Presentado

Aquí puedes ver cuánto debes declarar en cada modelo basándose en tus facturas y ventas.

### Histórico

El Histórico muestra todas las operaciones registradas:

- Lista de todas las facturas subidas
- Lista de todas las ventas importadas
- Filtros por fecha, proveedor, categoría
- Exportación de datos

### Crear Facturas

Permite generar facturas de venta a tus clientes:

1. Introduce los datos del cliente (nombre, NIF, dirección)
2. Añade los conceptos de la factura
3. Especifica importes y tipo de IVA
4. Genera el PDF de la factura
5. Descarga o envía directamente al cliente

### Consulta con IA

La sección de Consulta permite hacer preguntas sobre tus datos en lenguaje natural:

Ejemplos de consultas:
- "¿Cuánto gasté en software este año?"
- "Dame las facturas de Amazon de los últimos 3 meses"
- "¿Cuál es mi proveedor con más facturas?"
- "Resumen de gastos por categoría"

La IA analizará tus datos y te dará respuestas en formato de tabla, texto o gráfica.

## Preguntas Frecuentes

### ¿Cómo cambio los datos de mi empresa?
Contacta con soporte para modificar los datos de tu perfil de empresa.

### ¿Puedo exportar mis datos?
Sí, desde el Histórico puedes exportar facturas y ventas en formato CSV.

### ¿Qué hago si la IA extrae mal los datos de una factura?
Puedes editar manualmente los datos antes de guardar la factura.

### ¿La aplicación presenta los modelos a Hacienda?
Actualmente la aplicación prepara los datos, pero la presentación debe hacerse manualmente en la sede electrónica de la AEAT.
